# Yobta/stores

A collection of tiny observable stores designed to move logic away from components and help write reactive applications.

- **Small**. 162–674 Bytes
- **Tree-shakable**.
- **ESM-only**.
- **Typescript**.

## Istallation

```
npm i @yobta/stores
```
